# multi_array

pub/sub array data (float64) 
this repo's data is 

```
[0, 1, 2, 3]
```

# Usage
## python

- publisher 

```
rosrun multi_array pub
```

- subscriber

```
rosrun multi_array pub
```

## c++

- publisher 

```
rosrun multi_array pub
```

- subscriber 

```

rosrun multi_array sub
```
